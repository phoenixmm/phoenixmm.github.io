Android 系统新特性：5.0

Android 4.0
1 Space 留白
2 PopupMenu
3 GlidLayout 网格布局
 
Android 5.0

1    MaterialDesign设计风格
2    Material Theme
3    Meterial Dialog
4    CardView
5    RecyclerView
6    SwipeRefreshLayout
7    Toolbar
8    RippleDrawable 视图的水波纹效果
9    android L 增加了一些 Activity 的转场动画 —— 爆炸、滑动、淡入淡出
10    tint 前后背景着色
11    Palette 从图像中提取突出的颜色，这样可以把色值赋给 ActionBar、Toolbar、或者其他，可以让界面整个色调统一。
12    支持64位ART虚拟机
13    Heads-Up 风格通知。全新的通知中心设计（在锁屏界面也可以直接查看通知消息了，用户还可以直接在锁屏的情况下就行回复或进入应用。）
14    setTaskDescription 最近使用。全新的“最近应用程序”。除了界面风格设计的改变之外，新的最近应用界面还借鉴了 Chrome 浏览器的理念，采用单独的标签展示方式。更重要的是，谷歌已经向开发者开放了 API，所以第三方开发人员可以利用这个改进为特定的应用增加全新的功能。
15    Project Volta 电池续航改进计划。增加了 Battery Saver 模式，在低电量的时候系统会自动降低屏幕亮度、限制自动更换背景等功能。
16    MediaProjection 截屏
17    JobScheduler 通过为系统定义要在以后的某个时间或在指定的条件下（例如，当设备在充电时）异步运行的作业来优化电池寿命
18    setClipToOutline 裁剪
19    支持多种设备（手机、平板电脑、笔记本电脑、智能电视、汽车、智能手表甚至是各种家用电子产品）
20    新的 API 支持，蓝牙 4.1、USB Audio、多人分享等其它特性
21    Camera2
22    PdfRenderer 使用位图来呈现 PDF 文件
23    android.app.usage API 获取应用使用情况统计信息的 API
